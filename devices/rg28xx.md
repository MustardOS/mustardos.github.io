# Anbernic RG28XX H

## General Information

|:-------------------|:-------------------|
| Supported | Yes |

### Wifi Support

Currently Wi-Fi support is only via USB adapters using to 8188eu chipset.  
To enable it, use the **`Enable Wi-Fi (8188eu)`** option in the **Task Toolkit**.
